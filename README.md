# Путешествие по России

### Описание проекта:
Одностраничный сайт о путешествии по России. Учебный проект по верстке от Яндекс Практикума. В проекте испольсованы:
* Флексбоксы, гриды.
* Адаптивные элементы.
* Бэм-методология.

Сайт адаптирован под все виды устройств.
